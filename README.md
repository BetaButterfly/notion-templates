# Україномовні шаблони для Notion

## Використання
- Відкрийте шаблон через посилання «Перегляд».
- У Notion виконайте `/embed` і вставте URL — шаблон відобразиться на сторінці.
- Або завантажте `.html` і використовуйте локально.
