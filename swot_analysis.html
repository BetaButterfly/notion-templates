Сильні сторони (Strengths)
Внутрішні переваги — те, що ви робите добре
🏆 Конкурентні переваги
Що відрізняє вас від конкурентів?
🔧 Ресурси та можливості
Унікальні технології / патенти / бренд / команда
📊 Ринкова позиція
Частка ринку, лояльність клієнтів, репутація
💰 Фінансові показники
Рентабельність, стабільний грошовий потік, рейтинг
⚡ Операційна ефективність
Процеси, швидкість доставки, якість продукту/сервісу
Питання для роздумів:
Що ви вмієте краще за інших?
Які ресурси є унікальними для вас?
Чому клієнти обирають вас?
⚠️
Слабкі сторони (Weaknesses)
Внутрішні недоліки — що потребує покращення
🔒 Обмеження ресурсів
Брак фінансування, кадрів, технологій
⚙️ Операційні проблеми
Неефективні процеси, висока собівартість, якість
🎯 Прогалини в компетенціях
Відсутні навички, досвід, знання ринку
🔗 Залежності та ризики
Залежність від ключових постачальників / клієнтів
📢 Слабкий маркетинг / бренд
Низька впізнаваність, погана репутація, відгуки
Питання для роздумів:
Що можна покращити у продукті/команді?
Яких ресурсів чи навичок не вистачає?
Що клієнти вважають недоліком?
🚀
Можливості (Opportunities)
Зовнішні фактори, які можна використати на свою користь
📈 Ринкові тренди
Зростаючі сегменти, нові споживчі потреби
🌍 Нові ринки / географія
Регіони, країни, демографічні групи для розширення
🤝 Партнерства та альянси
Стратегічні партнери, дистриб'ютори, інвестори
⚡ Технологічні інновації
Нові технології, автоматизація, AI, платформи
📋 Регуляторні зміни
Нові закони, субсидії, зміни у конкуренції
Питання для роздумів:
Які тренди можна використати?
Які можливості пропускають конкуренти?
Чи є незадоволений попит на ринку?
🔥
Загрози (Threats)
Зовнішні фактори, що можуть зашкодити бізнесу
⚔️ Конкурентний тиск
Нові гравці, агресивне ціноутворення, замінники
🏢 Ринкові ризики
Зменшення попиту, зміна смаків споживачів
💱 Економічні фактори
Інфляція, курс валют, рецесія, зміна витрат
📄 Регуляторні ризики
Нові обмеження, штрафи, зміна законодавства
🔒 Технологічні загрози
Кібератаки, застаріння технологій, дизраптори
Питання для роздумів:
Що роблять конкуренти, що загрожує вам?
Які зовнішні зміни можуть зашкодити?
Які ваші слабкості найбільш вразливі?
🎯
Матриця стратегій (TOWS)
На основі SWOT сформулюйте стратегічні дії:
SO — Зростання
Сила + Можливості → атакуй
WO — Розвиток
Слабкість + Можливості → усунь прогалини
ST — Захист
Сила + Загрози → захищайся
WT — Виживання
Слабкість + Загрози → мінімізуй ризики
